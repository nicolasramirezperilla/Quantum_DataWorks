# <h1 align=center> **Project: Quantum DataWorks** </h1>

---

## <h2 align=center> `Description` </h2>

**Quantum DataWorks** is a project focused on automating and optimizing information processes at CVN, designed to enhance efficiency, reduce errors, and lower operational costs by implementing a robust and accurate data processing system. Using Python, pandas, and other tools, the project centers on creating a script to handle large volumes of data for generating detailed reports and analyses.

### <h3 align=center> `Script Functions` </h3>

1. **`proyecto_separacion(input_path, output_path)`**
   - **Description:** Processes an Excel file from `input_path` to separate and unify merchandise descriptions into a new file located at `output_path`.
   - **Technologies Used:** Python, pandas, openpyxl
   - **Impact:** Optimizes data organization and presentation, facilitating analysis and reducing errors in data entry.

2. **`proyecto_contador_reemplazos(df, columns_to_replace, replacement_value)`**
   - **Description:** Counts and replaces occurrences of `replacement_value` in specified columns of a DataFrame `df`, updating the DataFrame with new values.
   - **Technologies Used:** Python, pandas
   - **Impact:** Facilitates data normalization by replacing specific values across multiple columns, improving data consistency and quality.

3. **`proyecto_eliminar_filas_vacias(df)`**
   - **Description:** Removes empty rows from DataFrame `df`, ensuring that only rows with valid data are retained.
   - **Technologies Used:** Python, pandas
   - **Impact:** Cleans the DataFrame of empty entries that could affect subsequent analysis and processing.

4. **`proyecto_actualizar_columnas(df, date_column)`**
   - **Description:** Updates the columns of DataFrame `df` based on the `date_column`, adjusting data according to the specified dates.
   - **Technologies Used:** Python, pandas
   - **Impact:** Ensures data is up-to-date according to relevant dates, facilitating time-based analysis.

5. **`proyecto_calcular_diferencias(df, reference_column)`**
   - **Description:** Calculates differences between values in columns of DataFrame `df` compared to a `reference_column`.
   - **Technologies Used:** Python, pandas
   - **Impact:** Enables analysis of variations and trends by comparing data with a reference column.
